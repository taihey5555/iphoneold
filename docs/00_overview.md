# 中古スマホ監視bot 要件整理

## 目的
- 中古スマホ出品を監視し、条件抽出・粗利推定・危険フラグ付け・Telegram通知を行う。
- 自動購入は実装しない。
- 規約回避・保護回避機能は実装しない。

## 対象機種
- iPhone 13 128GB
- iPhone 14 128GB
- iPhone 15 128GB

## 処理フロー
1. 一覧ページ取得
2. 商品詳細取得
3. 正規化抽出
4. 除外ルール適用
5. 粗利計算
6. リスク閾値判定
7. 重複通知チェック
8. Telegram通知

## 非機能
- 取得は過剰アクセス回避の rate limiting を適用
- 設定は `.env` と `config.yaml`
- 保存は SQLite
- 実行は CLI ベース
